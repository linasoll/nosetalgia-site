«Nosetalgia» – парфюмерный бренд, изготавливающий по индивидуальному заказу ароматы, связанные с воспоминаниями.

Многостраничный сайт дает пользователям возможность записаться на консультацию, необходимую для создания аромата, получить актуальную информацию об акциях и мероприятиях и записаться на них, а также приобрести мерч.
